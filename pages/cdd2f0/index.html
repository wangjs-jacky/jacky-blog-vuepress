# 源码系列

# React 源码

  • 首推卡颂大佬的视频课和教程

    • 魔法师卡颂：

      曾给 React 贡献过源码，目前很多教程我看好多出自都是卡颂的React 技术揭秘这本电子书。

      • 《React 技术揭秘》 (opens new window)在线电子书
      • 视频课程：《自顶向下学React源码 (opens new window)》

    • 全栈潇晨

      大佬，教材总结的不错，视频课程除了听不懂其他的都挺好的。

      • 在线电子书：react 源码解析 (opens new window)

      • 视频课程：地址 (opens new window)

# Underscore 源码

  • underscore 官方源码解读 (opens new window)：可直接在控制台对代码进行调试，太方便了。
  • 推荐：淘系前端：冴羽-underscore 系列八篇文章。

# webpack源码

  • 《深入浅出Webpack (opens new window)》：很系统，但是缺点是大部分插件以及配置过时。
  • 【掘金手册】webpack5 核心原理与应用实践 (opens new window)：字节-范文杰大佬，观点和材料都很新，引经据典很好的梳理工作。
  • 推荐：饥人谷-webpack源码专精 (opens new window)

# Umi 源码

    1. 目前最好的教程肯定是：云谦大佬的极简框架课，但是知识星球的价格实在是太贵了。
    1. 掘金有一位大佬聪小陈 (opens new window) 开启日更 umi 节奏。

# Promise 源码

  • [推荐] 饥人谷手写 Promise 课程，课程链接 (opens new window)
    • 👍🏻这门课程是 TDD 基于测试驱动的开发模式，参考标准 PromiseA+，使用 mocha 和 chai 作为测试框架，可以在完全不了解 Promise 运行机制的情况下，只需要满足符合的测试案例即可完成一个标准的 Promise 库。

# JavaScript 基础

    1. 首推 MDN (opens new window)，大部分函数也给出 polyfill 实现，非常有助于底层函数实践。
    1. 阮一峰的《 ES6标准入门 (opens new window)》是学习 ES6 最好的教材，还有就是他的《网道》 (opens new window) 网站几乎涵盖了前端所有的内容，可以作为非常不错的新手入门教材。
    1. 《现代JavaScript教程 (opens new window)》精品，强推。

# VSCode 插件系列

  • 【推荐】微软的官方文档

    • Extension Guides (opens new window) : 对各功能模块的实践都做了详尽的导读。

    • API查询手册 (opens new window)：详细的函数使用手册。

    • 插件中一部分功能可通过配置 package.json 实现，与之有关的文档有

      • Contribution Points (opens new window)：核心字段编写查询，通过配置可实现如图标、欢迎面板、菜单面板、指令监听等功能。

      • Activation Events (opens new window)：监听事件，常用的也就是 onLanguage、onCommand、onUri、onWebviewPanel

      • Built-in Commands (opens new window)：与 vscode.commands.executeCommand 进行联动。

      • when-clause-contexts (opens new window)：可实现工程类型感知、场景识别等功能。
