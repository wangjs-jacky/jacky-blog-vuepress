DOM-根结点document对象

# DOM-根结点document对象

# HTML DOM 节点

在 HTML DOM (Document Object Model) 中 , 每一个元素都是 节点:

  • 文档是一个文档节点。
  • 所有的HTML元素都是元素节点。
  • 所有 HTML 属性都是属性节点。
  • 文本插入到 HTML 元素是文本节点。are text nodes。
  • 注释是注释节点。

# Document 对象

当浏览器载入 HTML 文档, 它就会成为 Document 对象。

Document 对象是 HTML 文档的根节点。

Document 对象使我们可以从脚本中对 HTML 页面中的所有元素进行访问。

**提示：**Document 对象是 Window 对象的一部分，可通过 window.document 属性对其进行访问。

# 浏览器支持

所有主要浏览器都支持 Document 对象。

# Document 对象属性和方法

HTML文档中可以使用以下属性和方法:

属性 / 方法 描述
document.activeElement (opens new window) 返回当前获取焦点元素
document.addEventListener() (opens new window) 向文档添加句柄
document.adoptNode(node) (opens new window) 从另外一个文档返回 adapded 节点到当前文档。
document.anchors (opens new window) 返回对文档中所有 Anchor 对象的引用。
document.applets 返回对文档中所有 Applet 对象的引用。注意: HTML5 已不支持 元素。
document.baseURI (opens new window) 返回文档的绝对基础 URI
document.body (opens new window) 返回文档的body元素
document.close() (opens new window) 关闭用 document.open() 方法打开的输出流，并显示选定的数据。
document.cookie (opens new window) 设置或返回与当前文档有关的所有 cookie。
document.createAttribute() (opens new window) 创建一个属性节点
document.createComment() (opens new window) createComment() 方法可创建注释节点。
document.createDocumentFragment() (opens new window) 创建空的 DocumentFragment 对象，并返回此对象。
document.createElement() (opens new window) 创建元素节点。
document.createTextNode() (opens new window) 创建文本节点。
document.doctype (opens new window) 返回与文档相关的文档类型声明 (DTD)。
document.documentElement (opens new window) 返回文档的根节点
document.documentMode (opens new window) 返回用于通过浏览器渲染文档的模式
document.documentURI (opens new window) 设置或返回文档的位置
document.domain (opens new window) 返回当前文档的域名。
document.domConfig 已废弃。返回 normalizeDocument() 被调用时所使用的配置。
document.embeds (opens new window) 返回文档中所有嵌入的内容（embed）集合
document.forms (opens new window) 返回对文档中所有 Form 对象引用。
document.getElementsByClassName() (opens new window) 返回文档中所有指定类名的元素集合，作为 NodeList 对象。
document.getElementById() (opens new window) 返回对拥有指定 id 的第一个对象的引用。
document.getElementsByName() (opens new window) 返回带有指定名称的对象集合。
document.getElementsByTagName() (opens new window) 返回带有指定标签名的对象集合。
document.images (opens new window) 返回对文档中所有 Image 对象引用。
document.implementation (opens new window) 返回处理该文档的 DOMImplementation 对象。
document.importNode() (opens new window) 把一个节点从另一个文档复制到该文档以便应用。
document.inputEncoding (opens new window) 返回用于文档的编码方式（在解析时）。
document.lastModified (opens new window) 返回文档被最后修改的日期和时间。
document.links (opens new window) 返回对文档中所有 Area 和 Link 对象引用。
document.normalize() (opens new window) 删除空文本节点，并连接相邻节点
document.normalizeDocument() (opens new window) 删除空文本节点，并连接相邻节点的
document.open() (opens new window) 打开一个流，以收集来自任何 document.write() 或 document.writeln() 方法的输出。
document.querySelector() (opens new window) 返回文档中匹配指定的CSS选择器的第一元素
document.querySelectorAll() (opens new window) document.querySelectorAll() 是 HTML5中引入的新方法，返回文档中匹配的CSS选择器的所有元素节点列表
document.readyState (opens new window) 返回文档状态 (载入中……)
document.referrer (opens new window) 返回载入当前文档的文档的 URL。
document.removeEventListener() (opens new window) 移除文档中的事件句柄(由 addEventListener() 方法添加)
document.renameNode() (opens new window) 重命名元素或者属性节点。
document.scripts (opens new window) 返回页面中所有脚本的集合。
document.strictErrorChecking (opens new window) 设置或返回是否强制进行错误检查。
document.title (opens new window) 返回当前文档的标题。
document.URL (opens new window) 返回文档完整的URL
document.write() (opens new window) 向文档写 HTML 表达式 或 JavaScript 代码。
document.writeln() (opens new window) 等同于 write() 方法，不同的是在每个表达式之后写一个换行符。
